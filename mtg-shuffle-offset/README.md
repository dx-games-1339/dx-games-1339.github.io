# MTG Shuffle Offset

A visual restitution of a Magic: The Gathering shuffle offset simulator. This is a placeholder template for UI and structure; game mechanics are not yet implemented.

## How to run

Open `index.html` in a browser or host this folder on GitHub Pages at:
`https://dx-games-1339.github.io/mtg-shuffle-offset/`

## File structure

- `index.html`: UI skeleton and controls.
- `style.css`: styling theme for board, controls, and overall layout.
- `script.js`: placeholder initialization and TODO plan.
